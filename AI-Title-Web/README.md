# AI-Title-Web

AI-Title 前端项目

## 项目部署

### 依赖安装

```
npm install
```

### 开发项目
```
npm run serve
```

### 构建项目
```
npm run build
```

### 语法检查
```
npm run lint
```

### 访问地址

Local: http://localhost:8080/
